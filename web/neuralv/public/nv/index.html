Package manager

NV

Отдельный пакетный менеджер. Ставь, обновляй и удаляй пакеты одной командой.

Установить Пакеты 
$ nv install neuralv@latest
$ nv install nv@latest
$ nv uninstall neuralv

Пакеты

NV не захардкожен только под NeuralV. Сервер отдаёт версии и артефакты по имени пакета.

neuralv

GUI и CLI версии клиента.

nv install neuralv@latest

nv

Сам пакетный менеджер.

nv install nv@latest

Версии

Используется обычный semver: `0.21.1`, `1.2.1` и так далее.

nv install neuralv@1.3.1

Основные команды

Установка пакета 
nv install <package>@latest
Установка версии 
nv install <package>@<version>
Удаление 
nv uninstall <package>
Текущая версия NV 
nv -v
