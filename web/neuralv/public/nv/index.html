Package manager

NV

Отдельный пакетный менеджер. Сначала ставишь NV, потом через него ставишь и обновляешь пакеты.

Установить Пакеты 
$ nv -v
$ nv install neuralv@latest
$ nv install nv@latest
$ nv uninstall neuralv

Установка NV

Выбери систему и получи готовый install flow.

Linux

Загрузка…
Команда 
curl -fsSL https://sosiskibot.ru/neuralv/install/nv.sh | sh
export PATH="$HOME/.local/bin:$PATH"
nv -v

Дальше

После установки NV можно сразу поставить NeuralV:

nv install neuralv@latest

Пакеты

NV ищет пакет по имени и берёт актуальные версии с сервера.

neuralv

Графическая версия приложения.

nv install neuralv@latest

nv

Сам пакетный менеджер.

nv install nv@latest

Версии

Используется обычный semver.

nv install neuralv@1.4.0

Основные команды

Установка или обновление 
nv install <package>@latest
Установка версии 
nv install <package>@<version>
Удаление 
nv uninstall <package>
Версия NV 
nv -v
