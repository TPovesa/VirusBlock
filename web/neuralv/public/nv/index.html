Package manager

NV

Один менеджер пакетов для Windows и Linux. Через NV можно ставить, обновлять и удалять приложения без ручной возни с архивами и длинными инструкциями.

Скачать Показать все пакеты 
$ nv -v
$ nv install package@latest
$ nv install nv@latest
$ nv uninstall package

Быстрый старт

Ставишь NV один раз, дальше работаешь с пакетами одной командой.

Один каталог

На одной странице видно доступные проекты, платформы и актуальные версии.

Прямое обновление

NV сам ставит свежие сборки, без ручной проверки архивов и ссылок.

Как это работает

Три коротких шага

01

Ставишь NV

Открываешь страницу загрузки и берёшь готовую команду под свою систему.

02

Выбираешь пакет

Открываешь каталог и смотришь, какие приложения есть для Windows и Linux.

03

Ставишь или обновляешь

NV скачивает актуальную версию и дальше обслуживает её той же командой.
